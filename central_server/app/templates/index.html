API Documentation

This document provides information about the AberOWL Central Server API, including the FAIR API (MOD-API).

Server Management API

POST /register

Register an ontology server. Returns a secret key for updates.

curl -X POST -H "Content-Type: application/json" -d '{"ontology": "MYONT", "url": "http://my.server.com/ontology"}' {{ request.url.scheme }}://{{ request.url.netloc }}/register
Query API

GET /api/servers

Get a list of all registered servers and their metadata.

Example: /api/servers

GET /api/search_all

Run a text search across all online servers.

Parameters: query (required), ontologies (optional, comma-separated).

Example: /api/search_all?query=cell

GET /api/dlquery_all

Run a DL query across all online servers.

Parameters: query (required), type (required, e.g., subclass), ontologies (optional, comma-separated).

Example: /api/dlquery_all?query='has part' some nucleus&type=subclass

FAIR API (MOD-API)

Catalogue

GET /: Get information about the semantic artefact catalogue.

Example: /?format=jsonld

Records

GET /records: Get all catalog records.

Example: /records?format=jsonld

GET /records/{artefact_id}: Get a specific catalog record.

Example: /records/GO?format=jsonld (replace GO with a registered ontology)

Artefacts

GET /artefacts: Get all semantic artefacts.

Example: /artefacts?format=jsonld

GET /artefacts/{artefact_id}: Get a specific semantic artefact.

Example: /artefacts/GO?format=jsonld (replace GO with a registered ontology)

Distributions

GET /artefacts/{artefact_id}/distributions: Get distributions for an artefact.

Example: /artefacts/GO/distributions?format=jsonld (replace GO with a registered ontology)

GET /artefacts/{artefact_id}/distributions/latest: Get the latest distribution.

Example: /artefacts/GO/distributions/latest?format=jsonld (replace GO with a registered ontology)

Resources (Partially Implemented)

These endpoints provide information about resources within an artefact. Most will return empty collections.

GET /artefacts/{artefact_id}/resources: Example (replace GO with a registered ontology)

GET /artefacts/{artefact_id}/resources/classes: Example (replace GO with a registered ontology)

GET /artefacts/{artefact_id}/resources/properties: Example (replace GO with a registered ontology)

Search

GET /search: Search metadata and content.

Example: /search?q=gene&format=jsonld

GET /search/content: Search content only.

Example: /search/content?q=cell&format=jsonld

GET /search/metadata: Search metadata only.

Example: /search/metadata?q=biological&format=jsonld
