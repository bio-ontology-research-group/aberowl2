About AberOWL

AberOWL: A Framework for Ontology-Based Data Access

AberOWL is a framework designed to provide reasoning services for bio-ontologies, enabling computational access to the knowledge contained within them. It facilitates semantic access to biological data and literature by not just relying on class names or identifiers, but on the actual knowledge the ontologies contain and the inferences that can be drawn from it.

The original AberOWL system was described in the paper: Aber-OWL: a framework for ontology-based data access in biology, published in *BMC Bioinformatics* in 2015.

AberOWL 2: Development and Team

This version, AberOWL 2, represents a significant re-engineering of the original framework. It introduces a distributed architecture where individual ontology servers can be deployed independently and register with a central server. This enhances scalability, maintainability, and flexibility.
